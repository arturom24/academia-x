Las etiquetas de HTML semántico son aquellas que se utilizan para proporcionar significado y estructura al contenido de una página web, facilitando la comprensión del mismo tanto para los desarrolladores como para los navegadores y los motores de búsqueda. A continuación, se presentan algunas de las etiquetas semánticas más comunes en HTML5:

  1. <header>: Define la cabecera de un documento o una sección.
  2. <nav>: Define una sección de navegación en la página.
  3. <main>: Define el contenido principal de un documento.
  4. <article>: Define un contenido independiente que puede ser reutilizable o distribuible por separado del resto del contenido.
  5. <section>: Define una sección temática en un documento.
  6. <aside>: Define contenido secundario o relacionado con el contenido circundante, como barras laterales, notas y elementos relacionados.
  7. <footer>: Define el pie de página de un documento o una sección.

Estas etiquetas, junto con otras como <div> y <span>, son parte de HTML5 y permiten una mejor organización y estructuración del contenido web, lo que beneficia tanto a los desarrolladores como a los usuarios finales. Además de estas etiquetas semánticas, HTML5 también introdujo etiquetas específicas para elementos multimedia, formularios y otras funcionalidades, lo que mejora la accesibilidad y la experiencia de usuario en general.
